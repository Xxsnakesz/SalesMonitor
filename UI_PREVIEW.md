### Action Buttons

**Primary Actions:**
- **Create User** (Top right, blue button)
- **Edit** (Blue text link)
- **Reset Password** (Yellow text link)
- **Delete** (Red text link)

**Modal Actions:**
- **Create User / Update User** (Blue button)
- **Reset Password** (Blue button)
- **Delete** (Red button)
- **Cancel** (Gray button)

## Responsive Design

### Desktop (> 768px)
- Full table visible
- All columns shown
- Side-by-side stats cards
- Wide modals

### Tablet (768px - 1024px)
- Table scrollable horizontally
- Stats cards in 2x2 grid
- Medium modals

### Mobile (< 768px)
- Horizontal scroll for table
- Stats cards stacked vertically
- Full-width modals
- Touch-friendly buttons

## Interactive States

### Hover States:
- Table rows: Light gray background
- Buttons: Darker shade
- Links: Underline appears

### Loading States:
- Buttons show "Saving..." / "Deleting..." / "Resetting..."
- Disabled state (opacity: 50%)
- Cursor: not-allowed

### Error States:
- Red border on invalid fields
- Red error text below field
- Error banner at top of form
